# Hit and Blow
## 制作アプリの説明
- オフライン戦/オンライン戦ができるHit and Blowゲーム

## 工夫した点・こだわった点
- Realtime databaseにより、リアルタイム対戦を実現
- 観戦モードも設け、対戦の様子をチャットしながら観戦できる
- アイテム機能により、戦略的なゲーム展開が可能に
- 実況機能により、チャット欄が盛り上がる！

## 次回トライしたいこと
